# Viinikanta

Tämä on Helsingin yliopiston Tietokantasovellus harjoitustyökurssin toteutus. Työssä rakennetaan verkkosivusto, jolla voi arvioida viinejä ja tallentaa arvioinnit tietokantaan.

Toimintoja:
    -Käyttäjätunnusten luonti
    -Viinien luonti
    -Viinien arvostelu
    -Viinien selaaminen
    -Viinien poisto
